✏️ Template mẹo tùy chỉnh
📝
Viết mẹo theo format JSON dưới đây. Nhấn Xem trước để kiểm tra rồi Thêm vào trang Mẹo.
JSON template:
Preview:
Hướng dẫn schema JSON
{
  "id": "my-tip-001",           // unique ID
  "title": "Tên mẹo",           // hiển thị trong mục lục
  "icon": "🚀",                  // icon (emoji)
  "badge": "Custom",             // nhãn badge (tùy chọn)
  "color": "#3d8ef8",            // màu border-top
  "type": "text|table|rules|speeds",  // kiểu hiển thị
  "content": [                   // nội dung (tùy type)
    { "text": "..." },           // type=text: đoạn văn
    { "html": "..." },           // type=text: raw HTML
    { "insight": "...", "class": "green" }  // insight box
  ]
}
// type=table: thêm "headers":[], "rows":[[...]]
// type=rules: thêm "cards":[{"title","color","items":[]}]
// type=speeds: thêm "items":[{"val","unit","desc"}]
